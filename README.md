# SharePoint Embedded for Visual Studio Code (Preview)
The SharePoint Embedded Visual Studio Code extension helps developers get started with SharePoint Embedded application development. With the extension, developers can:

1. Create and configure Azure Entra app registrations
1. Create and manage Free Trial Container Types
1. Create additional guest apps on a Container Type
1. Load one of the sample apps and populate its runtime configuration
1. Export Container Type and Azure Entra app settings to a Postman Environment file for use with the SharePoint Embedded Postman Collection
